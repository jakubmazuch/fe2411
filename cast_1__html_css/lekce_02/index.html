Definiční seznamy

Mrkev
Mrkev je zelenina, která je oranžová a chutná sladce.
Brambory
Brambory jsou hlízovitá zelenina, která je bílá a chutná neutrálně.
Celer
Celer je zelenina, která je zelená a chutná hořce.
Rajče
Rajče je zelenina, která je červená a chutná kyselo.
Ředkvičky
Ředkvičky jsou zelenina, která je červená a chutná ostružinově.
